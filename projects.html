# 👨‍💻我的專案

# 前言

能夠從頭到尾打造一個電子遊戲長久以來一直是我的夢想, 自從發現了 Udemy 這個線上學習頻道打開了我的視野: 任何人都可以在 Udemy 授課, 將知識和技術分享給全世界所有人, 而且課程售價非常親民, 吃一餐好料的費用就可以買到一門長達 20 幾小時的課程, 非常划算. 以下的遊戲是 Udemy 課程的範例專案, 都是 HTML5 遊戲, 開發工具有 Phaser.js Three.js 等

# 使用 Phaser.js 框架開發 2D 網頁遊戲

Phaser 是一款很受歡迎的 HTML 遊戲引擎 / 框架, 可以快速產生 2D 的遊戲. 因為用 原生的 JavaScript Canvas 開發遊戲工程浩大, 而且況日費時, 所以許多精通 JavaScript 的開發者把繁雜的程序用一個簡單的函式 / API 包裝起來, 讓遊戲開發者能輕鬆容易的製作出遊戲. 以下是我在 Udemy 購買的教學課程中的專案範例, 都是英文教學, 很可惜沒有中文的, 我想或許以後我來開個中文課程吧. Phaser 最新版本是 3.x, API 變動滿大的, 主要因應 ES6 的強勢來襲, 幾乎所有的開發方式都採用 class 模組化的方式編程, Phaser 也不例外.
