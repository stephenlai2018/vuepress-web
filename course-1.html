# 第 1 節 : 綁定資料為一個元素的 class

這是甚麼意思呢? 這個範例中是把 html 標籤的 class 屬性和 Vue 實例 computed 的屬性做動態綁定, 例如 :

<button :class="btnClass">Submit</button>

btnClass 是 Vue 實例 computed 的屬性, 可以是字串, 例如 :

btnClass() {
  if (this.text == 0) { 
    return 'submit disabled'
  } else {
    return 'submit'
  }
}

也可以是物件, 例如 :

btnClass() {
  if (this.text == 0) { 
    return {
      submit: true,
      disabled: ture
    }
  } else {
    submit: true,
    disabled: false
  }
}

物件的進階寫法 :

btnClass() {
  return {
    submit: true,
    disabled: this.text.length == 0
  }
}

# 第 2 節 : 綁定資料為一個元素的 style

這是甚麼意思呢? 其實就是把 html 標籤的 style 屬性和 Vue 實例做動態綁定, 例如 :

<h1 :style="h1Style">Valar Morghulis</h1>

h1Style 是 Vue 實例 data 的屬性, 可以是字串, 例如 :

data: {
  h1Style: 'color: red'
}

可以是物件, 例如 :

data: {
  h1Style: {
    color: red'
  }
}

可以是陣列 (陣列內必須是物件!!!) :

data: {
  h1Style: [
    {
      color: red'
    },
    {
      fontSize: '20px'
    }
  ]
}

陣列的另一種表達方式 :


const commonHeadStyle = {
  fontSize: '20px' // camelCase
}

const greenWord = {
  color: 'green'
}

data: {
  h1Style: [
    commonHeadStyle,
    greenWord
  ]
}
